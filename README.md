# Шляпа

Веб-игра в Шляпу для компании друзей. Работает на смартфоне — один телефон передаётся по кругу.

## Локальный запуск

Нужен установленный Node.js (версия 18 или новее).

```bash
npm install
npm run dev
```

Откроется на http://localhost:5173.

## Деплой на GitLab Pages

Сайт автоматически публикуется при каждом пуше в основную ветку. Один раз настраиваем — дальше работает само.

### Шаги

1. **Создай новый проект на GitLab** (можно публичный, можно приватный — Pages работают и с приватными).

2. **Залей этот проект в свой репозиторий:**
   ```bash
   cd hat-game
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://gitlab.com/<твой-username>/<имя-репозитория>.git
   git push -u origin main
   ```

3. **Дождись окончания CI/CD.** В разделе **Build → Pipelines** должен появиться job `pages`. Он соберёт сайт за пару минут.

4. **Найди адрес сайта:** в разделе **Deploy → Pages** будет ссылка вида:
   ```
   https://<username>.gitlab.io/<имя-репозитория>/
   ```

Готово. Этот адрес можно скинуть друзьям, открыть на любом смартфоне, добавить на главный экран.

### Если что-то не задеплоилось

- Проверь, что у проекта включены **Pipelines** (Settings → CI/CD → General pipelines).
- Если ветка называется `master`, а не `main` — ничего менять не нужно, CI ловит обе.
- Если хочешь свой домен — настраивается в Deploy → Pages → New domain.

## Установка на смартфон как приложение

После открытия сайта на телефоне:

- **iPhone (Safari):** кнопка «Поделиться» → «На экран Домой».
- **Android (Chrome):** меню (три точки) → «Добавить на главный экран».

Иконка появится на рабочем столе, открываться будет как обычное приложение, без адресной строки.

## Как менять словарь

Слова лежат в `src/words.js` — три массива по уровням сложности. Допиши свои, сохрани, запушь — деплой сам обновится.

```

```

## Как играть

1. Введи имена игроков (минимум 4, чётное число).
2. Выбери сложность словаря, количество слов на игру и длительность раунда.
3. Жми «Начать игру» — игроки автоматически разобьются на пары.
4. Дальше игра проводит по раундам: на экране — кто объясняет, кто угадывает, и большая кнопка «Поехали».
5. Во время раунда: объясняющий смотрит на слово и объясняет его напарнику. При угаданном слове — нажимает «Угадал». Если объяснить не получается — «Закончить раунд».
6. По кругу пары меняются ролями, пока в шляпе не кончатся слова.
7. В конце — таблица результатов.

Прогресс сохраняется в браузере — если страницу случайно закрыть, можно вернуться и продолжить с того же места.
